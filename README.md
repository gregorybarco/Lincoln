# Lincoln

A local AI agent built on open source tools with zero cloud dependency.

## Purpose
- Local AI agent with knowledge up to 2025 chosen as qwen model and not gemma for its reasoning capabilities.
- Agentic assistance for mathematical finance research project

## Launch
' >lincoln '

## Interaction
/ask  - ask questions and get suggestions
/add  - add a file to Lincoln's context
/drop - remove a file from context
/exit - close Lincoln

## Stack
- Ollama — local model serving
- Qwen 3.5 9B — reasoning and agent model
- Aider — agentic code assistance layer
- Python 3 — pipeline and tooling language

## Status
Operational. RAG pipeline in planning.
